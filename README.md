## Утилита полностью автоматизирует процесс сохранения плейлиста. Вам нужно только вставить ссылку, а всё остальное AsbestDLP берет на себя:  
```
>_ Максимальное качество: Извлечение аудиопотока в формате MP3 с наилучшим доступным битрейтом.

>_ Умные обложки: Автоматический поиск, конвертация в JPEG и «вшивание» арт-ворка прямо в файл.

>_ Идеальные метаданные: Программа сама заполняет теги (исполнитель, название).

>_ Все треки бережно складываются в папку Music (C:\Users\user\Music)
```

## AsbestDLP является модифицированным форком оригинального yt-dlp. Мы проанализировали главные «боли» пользователей оригинального софта и внедрили решения из коробки:
```
>_ Оригинальный софт часто оставляет в названии ID видео (вроде [h3mfk3k6]) и лишний текст типа
   "(Official Video)". AsbestDLP имеет встроенный фильтр, который вырезает этот хлам на лету.

>_ Оригинал часто сохраняет обложки в формате .webp, который не видят старые плееры, или оставляет их отдельными файлами в папке.
   Наш форк принудительно конвертирует их в совместимый JPG и вшивает внутрь трека, удаляя временные исходники.

>_ Вам больше не нужно возиться с командной строкой и сложными флагами. Все нужные параметры уже зашиты в AsbestDLP.
```

## Из коробки в AsbestDLP доступно быстрое создание плейлиста из папки с вашей музыкой в Медиаплеере Microsoft:
```
>_ Вам не придется вручную перетаскивать сотни песен в Media Player. Просто укажите путь к папке
   с вашими треками, и утилита мгновенно создаст готовый файл .m3u8.

>_ Вы сами управляете порядком воспроизведения — создавайте плейлисты по дате добавления
   (сначала самые свежие новинки) или в классическом хронологическом порядке.
```
